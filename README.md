# Deed Training Backend (Regenerated)

This repository contains the backend for the Deed Training Platform.

## Quick start (Docker)

1. Copy `.env.example` to `.env` and edit values.
2. Start Postgres and app (see docker-compose or use a managed Postgres).
3. Run migrations (migrations/001_init.sql) against your Postgres.
4. npm install
5. npm start

API endpoints are under `/api/*`.
